# Rick & Morty API - Project Overview

This repository contains a full implementation of a **GraphQL API for Rick & Morty characters**, created as part of a technical test.  

All the code, configurations, and scripts are inside the `rickmorty-api` folder.

## Project Overview

- **API Type:** GraphQL
- **Features Implemented:**
  - CRUD operations for characters.
  - Data synchronization with the official [Rick & Morty API](https://rickandmortyapi.com/).
  - Database caching with Redis.
  - Sequelize ORM with PostgreSQL.
  - Cron jobs to fetch and update character data automatically.
  - Database migrations and seeders included.
- **Folder Structure:** All backend code, Docker configuration, and scripts are located inside `rickmorty-api`.
- **ERD:** Database model diagram is available inside the repository as `ERD.png`.

## How to Run

You can run the project either **with Docker** or **manually**.  
For detailed instructions, check the README inside `rickmorty-api`.
